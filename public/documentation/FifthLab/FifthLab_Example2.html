1. Настройка агента
    • Проверьте, что созданный агент появился в списке агентов.

    Теперь нужно соединить новый порт (NodeSochi) с существующими морскими путями:

    1. Дважды кликните левой кнопкой мыши по краю узла.
    2. Проведите линию (путь) к уже существующему морскому маршруту.
  2. Параметры порта

    3. Настройте свойства агента, используя пример на рисунке:

    • Connected Ports (Связанные порты): укажите, с какими другими портами будет связан ваш новый порт.
    • Capacity (Вместимость): задайте максимальное количество контейнеров, которое может обслуживать порт.
    • Location (Местоположение): убедитесь, что агент размещён в узле
    • Откройте свойства агентов, с которыми вы хотите соединить свой порт: Вкладка «Проекты» → «Container Line» → «Main» → «Агенты».
    • В поле Connected Ports добавьте название вашего порта (Sochi), чтобы установить связь.
  3. Моделирование
    • Для запуска модели нажмите F5.
