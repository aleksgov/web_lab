1. Настройка агента
    • Убедиться, что новый агент появился в списке агентов модели.
    • Соединить новый узел с морскими путями (палитра → соединитель).
  2. Параметры порта

    3. Настроить параметры исходя из рисунка:

    • Порты, с которыми соединен новый порт (Connected Ports).
    • Вместимость порта (Capacity).
    • Местоположение агента (Location) ‐ совместить точечный узел с агентом.
  3. Открыть свойства выбранных для соединения агентов и добавить в Connected Ports свой порт.
  4. Моделирование
    • Запустить модель, нажав F5.
