Руководство к пользованию «DataLab»

Веб-приложение «DataLab» представляет собой интерактивную платформу для изучения и выполнения лабораторного практикума курса «Управление данными».

Диалоговое окно приложения разделено на четыре основных блока:

  • Центральный блок
  • Навигационная панель
  • Кнопка информации
  • Меню выбора цвета фона

Центральный блок

В центре экрана расположен блок с пятью кнопками:

  • Лабораторная работа №1: «Моделирование. Язык программирования GPSS»
  • Лабораторная работа №2: «Язык разметки XML»
  • Лабораторная работа №3: «Автоматизация с использованием ZennoPoster»
  • Лабораторная работа №4: «Знакомство со средой Cisco Packet Tracer. Моделирование простой сети»
  • Лабораторная работа №5: «Моделирование в AnyLogic»

Навигационное меню

Навигационная панель расположена в верхней части экрана слева. Панель позволяет пользователю ориентироваться по разделам программы, предоставляя информацию о текущем расположении, например: "Главная → Лабораторная работа №2".

Центральный блок лабораторных работ

Во вкладке лабораторных работ в центре экрана расположен блок с тремя основными кнопками:

  • Теория: Кнопка открывает вкладку с теоретическими материалами по теме "Язык разметки XML", необходимыми для выполнения лабораторной работы.
  • Пример: Кнопка ведет на вкладку с практическим примером выполнения задания, демонстрирующим основные подходы к решению задач второй лабораторной работы.
  • Варианты заданий: Кнопка открывает вкладку с вариантами заданий. На данной вкладке пользователь может выбрать конкретный вариант, после чего его перенаправит на отдельную страницу с заданием.

После того как пользователь выберет конкретный вариант задания, ему откроется отдельная вкладка, где будет представлено само задание, описание задачи и инструкции по ее выполнению. Эта вкладка позволяет работать с выбранным вариантом задания в удобном формате, организуя пространство для выполнения лабораторной работы.

Кнопка информации

В правом нижнем углу экрана находится кнопка, предоставляющая доступ к инструкции по использованию программы. Нажатие на неё открывает раздел с подробным описанием функциональности и порядка работы с программой.

Меню выбора цвета фона

Меню позволяет пользователю изменить цветовую схему фона интерфейса, предоставляя выбор из нескольких вариантов.
