1. Определение единиц измерения: В GPSS одна единица времени равна одной минуте.
  2. Написание кода модели: Создайте новую модель «File > New > Model»
  3. Используйте следующие команды GPSS для описания работы парикмахерской:
  4. Пояснения к коду: 
    5.  GENERATE 25,7    ; Имитация случайного интервала между прибытием
                  ; клиентов 25 ± 7 минут
 QUEUE barberq    ; Учет очереди, в которой клиенты ожидают
 SEIZE barber     ; Если парикмахер свободен, клиент занимает
                  ; кресло
 DEPART barber    ; Клиент покидает очередь и занимает парикмахера
 ADVANCE 18,5     ; Имитация времени стрижки 18 ± 5 минут
 TERMINATE 0      ; Завершение транзакции
 RELEASE barber   ; Освобождение парикмахера после завершения
                  ; обслуживания
 TERMINATE 0      ; Завершение транзакции
 GENERATE 600     ; Завершение работы парикмахерской после
                  ; 10 часов (600 минут)
 TERMINATE 1      ; Остановка симуляции после завершения последнего
                  ; события
  5. Запуск программы: После написания модели запустите её действием «Command > Create Simulation»
