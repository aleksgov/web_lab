1. На рабочей области программы добавьте два коммутатора Switch-PT. По умолчанию они будут иметь имена Switch0 и Switch1.
  2. Добавьте на рабочее поле четыре компьютера с именами по умолчанию: PC0, PC1, PC2 и PC3.
  3. Соедините устройства в сеть Ethernet, как показано на рисунке 5.4.
  4. Сохраните созданную топологию, нажав кнопку "Save" в меню File → Save.

Рис.5.4. Экспериментальная модель сети

Далее откройте свойства устройства PC0, щелкнув на его изображение. Перейдите на вкладку Desktop и откройте Command Prompt. Введите команду «?» и нажмите Enter для получения списка доступных команд. Для конфигурирования IP-адреса компьютера используйте команду ipconfig. Например, для настройки IP-адреса 192.168.1.2 с маской подсети 255.255.255.0 выполните команду:

ipconfig 192.168.1.2 255.255.255.0

IP-адрес и маску сети также можно настроить через графический интерфейс устройства (см. рисунок 5.5). Параметр DEFAULT GATEWAY не требуется, так как создаваемая сеть не предполагает маршрутизации.

Рис.5.5. Настройка узла

Повторите этот процесс для каждого компьютера.

Таблица 2. Настроенные IP-адреса и маски подсети:

Устройство IP-адрес Маска подсети
PC0 192.168.1.2 255.255.255.0
PC1 192.168.1.3 255.255.255.0
