Задание

  1. Установить программное обеспечение

    Скачать и установить демонстрационную версию ZennoPoster с официального сайта: https://zennolab.com/ru/products/zennoposter/. Для выполнения задания достаточно версии с поддержкой одного потока.

  2. Ознакомиться с функционалом программы

    Использовать следующие ресурсы для изучения возможностей ZennoPoster:

    • Видеоуроки: YouTube-канал ZennoLab
    • Документация и база знаний: Официальный справочник ZennoPoster
  3. Разработать шаблон автоматизации

    Создать шаблон для автоматизации действий на любом выбранном сайте. Тип шаблона также определяется студентом:

    • Парсер
    • Авторегистратор (авторегер)
    • Постер

    Приветствуется разработка проекта на Post/Get запросах.
